# Visit Section Module

بخش ویزیت نرم افزار از کاربردی ترین ماژول های نرم افزار است.
امکان درج شرح حال، مشکل فعلی بیمار و تشخیص پزشک، دارو و Plan ، Order در ویزیت های مختلف و همچنین تشکیل favorite list و تشکیل پروفایل برای داروهای روتین وجود دارد.
بخش ویزیت نرم افزار شامل اطلاعات کامل مشخصات فردی بیمار، سوابق بیماری، سوابق بالینی، معاینات، انواع معاینات تخصصی، انواع پاراکلینیک‌های تخصصی و تشخیصات و تصمیمات پزشک، خلاصه پرونده و اطلاعات دستوری پزشک جهت ارائه خدمت به بیمار می باشد.
امکان مشاهده شرح حال بیمار در یک نگاه و قابلیت دریافت پرینت در این ماژول قرار داده شده است.
امکان ثبت گزارشات پاراکلینیکی هم در این ماژول وجود دارد.
همچنین از طریق ماژول گزارشات نرم افزار امکان دریافت خروجی تحقیقاتی excel بر اساس داده های پرونده و بخش ویزیت وجود دارد که با sort آن می توان به دیتای مهمی از پرونده های مطب دسترسی داشت و برای پرخی از پزشکان چنین گزارشاتی مهم است.
