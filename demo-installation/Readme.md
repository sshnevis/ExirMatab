# Demo Installation

دانلود برنامه از لینک زیر انجام دهید. 
https://exirmatab.com/uploads/Exir-latest.zip
فایل دانلود شده را در همان پوشه،Extract کرده و فایل ExirSetup.exe را اجرا کنید.
بر روی شروع نصب کلیک کرده و گزینه (نسخه دمو) را انتخاب کنید . محل ذخیره سازی پوشه را بر روی درایو دیگری بجز درایوC قراردهید. تیک گزینه نصب پایگاه داده بر روی این سیستم را هم فعال کنید و بر روی مرحله بعد کلیک کنید و منتظر بمانید فرآیند نصب برنامه کامل شود.
نکته: نصب پایگاه داده فقط بر روی سیستم سرور (اصلی) انجام شود.
نکته: در صورت استفاده از نسخه دمو و ایجاد پرونده بهتر است اطلاعات روی درایو C نباشد تا ریسک پاک شدن اطلاعات نداشته باشید.
پس از تکمیل شدن فرآیند نصب، بر روی ضربدر کلیک کرده و پنجره را ببندید.
پس از نصب، آیکون برنامه اکسیر بر روی Desktop قابل مشاهده هست، بر روی آن کلیک کرد و وارد برنامه شوید. 
برای ورود اولیه به برنامه، نیاز به نام کاربری و پسورد نیست، فقط بر روی وارد شوید کلیک کنید.
نکته: تا زمانی که در تنظیمات برنامه بخش کاربران، نام کاربری و رمز عبور تعیین نکنید با کلیک بر روی وارد شوید می‌توانید وارد برنامه شوید.
نسخه دمو اکسیر کاملا رایگان بوده و تا 100 پرونده بدون محدودیت زمانی قابل استفاده می باشد.
